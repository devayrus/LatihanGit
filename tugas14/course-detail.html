Web Development

Learn how to build websites and web applications from scratch using HTML, CSS, JavaScript, and more.

About this Course

This course will teach you the fundamentals of web development, including how to build and style websites using HTML, CSS, and JavaScript. You'll also learn about modern tools and frameworks like Bootstrap, React, and Node.js, enabling you to create responsive and dynamic web applications.

  • Duration: 8 weeks
  • Level: Beginner to Intermediate
  • Lessons: 17
  • Instructor: Albert Flores

Course Outline

  • Introduction to Web Development
  • HTML Basics
  • CSS Fundamentals
  • JavaScript Essentials
  • Responsive Design with Flexbox and Grid
  • Building Dynamic Websites with JavaScript
  • Version Control with Git and GitHub
  • Introduction to Bootstrap
  • Working with APIs
  • Introduction to React
  • Node.js and Express.js
  • Building Full-Stack Applications
